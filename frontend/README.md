# Redline Frontend

Frontend review hub for Redline compliance workflows.

## Stack

- React 18
- TypeScript
- Vite
- React Router
- React Query
- Zod

## Development

From repo root:

```bash
bun run dev:frontend
```

Or directly from this folder:

```bash
bun install
bun run dev
```

## Build

From repo root:

```bash
bun run build:frontend
```

Or directly from this folder:

```bash
bun run build
```

## Lint

From repo root:

```bash
bun run lint:frontend
```

Or directly from this folder:

```bash
bun run lint
```
